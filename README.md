### Project Setup

1. Get Docker Images
  ```
  docker pull lumialfe/carspottr-webapp
  docker pull lumialfe/carspottr-restapi
  ```

2. Start the project

  ```
  docker compose up --build
  ```

3. Shut down the project
  ```
  docker compose down
  ```
